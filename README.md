# Login & Registration System in C++

This is a simple Login and Registration System built using C++.  
It allows users to register and log in using credentials stored in a text file.

## 🔹 Features
- User Registration
- User Login Authentication
- File Handling using fstream
- Data stored in users.txt
- Menu-driven program

## 🔹 Technologies Used
- C++
- File Handling (ifstream, ofstream)

## 🔹 How to Run
1. Compile the code:
   g++ main.cpp -o main
2. Run the program:
   ./main

## 🔹 Project Purpose
This project was created as part of my internship to understand basic authentication and file handling in C++.
